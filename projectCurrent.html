TCoB - Taking Care of Business

App Description

A community-driven mapping tool designed to improve public access to essential civic amenities: restrooms, water fountains, trash receptacles, and pet waste bag stations. The app leverages Google Maps and verified user contributions to create a reliable, real-time resource for residents, visitors, and pet owners seeking clean and accessible public infrastructure. By integrating Google OAuth authentication, the platform ensures data integrity while empowering communities to report, confirm, and crowdsource the availability and condition of local facilities. The app supports cleaner, more navigable public spaces — with focus on civic responsibility.

Tech Overview

I’m building a location-based civic mapping app using React, Rails, the Google Maps API, and Google OAuth to allow verified users to report public restrooms, water fountains, waste bins, and pet waste bag stations.

The app will implement token-based authentication using OAuth2 and JWT, and feature a mobile-first UI designed with TailwindCSS. I plan to deploy the React frontend on Netlify and the Rails API backend on Render.

By combining geospatial data modeling with client-side state management, the app will deliver a responsive, real-time experience tailored to residents, tourists, and pet owners alike.

Tech In More Detail

Fullstack Architecture: A decoupled React frontend and Rails API backend communicating over RESTful endpoints.

Authentication: Google OAuth2 with secure JWT-based session handling for verified contributions.

Mapping: Uses the Google Maps JS API with Geolocation to center the map on the user’s location and display civic resources.

Geospatial Storage: Coordinates stored in a PostgreSQL-backed Rails API with filtering by type and proximity.

Data Modeling & CRUD: Models for users and resource types with full CRUD via the API and basic moderation logic.

Frontend Components: Modular React components for forms, filters, map markers, and stateful interactions.

Mobile-First Design: TailwindCSS for responsive layouts ideal for on-the-go use.

API Testing: HTTPie and Postman used throughout development for endpoint and auth flow testing.

Deployment: Rails backend on Render, React frontend on Netlify, with secure handling of API keys and secrets via environment variables.

User Experience Focus: Designed for urgency and ease — helping people find a restroom, trash bin, or water fountain with minimal friction.
